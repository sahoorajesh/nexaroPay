## Files

- `favicon-16.png`, `favicon-32.png`, `favicon-48.png`: standard favicon sizes

- `icon-192.png`, `icon-512.png`: common PWA icon sizes

- `favicon-dark-32.png`
- `icon-dark-192.png`
- `icon-dark-512.png`
